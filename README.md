# Cruise In Auto Repair

Production-quality demo website for Cruise In Auto Repair in Garden City, Michigan.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Development

```bash
pnpm install
pnpm dev
```

## Production Check

```bash
pnpm lint
pnpm build
```

The site is frontend-only and deploy-ready for Vercel.
